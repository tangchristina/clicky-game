# clicky-game

Clicky game is a Javascript application in which users test their memory by clicking on photos of cars. 
React.js is used to render the front-end and events are handled using set state. 
Simply click on the car image for your choice. The stack of cards will then shuffle around. Click on a different card and your score will increment by one. However, if the same card gets clicked, it will be game over and the score resets to 0.

 
Prereqs:

Npx create-react-app clicky-game

Npm install react

Npm install react-router-dom


Npm run start to open the app in the local browser



Components:

Card – displays car images

Navbar- displays scoreboard and navigation

Jumbotron – displays title and instructions

Wrapper – wraps card component
